# r-subsync fork

### Additions
--synclang {lang}
--check

### Usage
if movie has 3 langs You sync one with

`python3 rsubsync.py --lang en --path ../`

and others with

`python3 rsubsync.py --lang fr --path ../ --synclang en`

this reduces total sync time
